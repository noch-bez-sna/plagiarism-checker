Для запуска проекта запускать класс Launcher иначе не заработает JavaFX.

Сборка jar и логирование осуществляется в папку build.

Собранные jar и javadoc (javadoc --> index.html) лежат в корне проекта.

В корне проекта лежат две базы данных (big_database и small_database) выбрать можно любую. Также в корне проекта в папке test_programs лежит код для проверки, но воообще можно использовать .java файл.

Чтобы загрузить базу данных переходим во вкладку База данных --> Выбрать папку. После того как выбрали папку --> Загрузить базу. Появится сообщение об успешной загрузке

Код можно проверить 2-мя способами. 
Первый: Проверка файла --> Обзор --> Выбираем .java файл --> Анаизировать
Второй: Копируем код в поле "Вставьте Java код здесь" --> Проверить код

В поле результаты мы видим название файла из базы данных, процент совпадения и количество совпадений.
Более подробные результаты о конкретных совпадениях с каждым файлом из бд представлены ниже. Чтобы переключаться между ними просто кликаем по нужному файлу в поле выше, с общими результатами.

Исползуемые версии (по идее для сборки достаточно установить Java 21 и всё заработает):

    Java: Microsoft OpenJDK 21.0.9 (устанавливается через IntelliJ IDEA, но также подойдёт любая версия Java 21)

    Gradle: Версия 8.5 (Wrapper) 
    
    Gradle JVM: Project SDK ms-21

    JavaFX: Версия 21.0.2

    JavaParser: Версия 3.25.4

    Log4j 2: Версия 2.20.0

    JUnit 5: Версия 5.9.
